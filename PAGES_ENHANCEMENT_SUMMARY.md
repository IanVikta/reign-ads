## 🎨 Design System Consistency

### Colors Used:
- `reignBlack`: #111111 (backgrounds)
- `reignGold`: #D4AF37 (primary accent)
- `reignOrange`: #FF8C00 (secondary accent)
- `charcoal`: #333333 (secondary backgrounds)
- Gradient: from-reignGold via-yellow-400 to-reignGold

### Typography:
- Display font: Outfit (for headlines)
- Body font: Inter (for text)
- Mono font: for labels and tags

### Common Patterns:
- Background patterns with radial gradients
- Border effects with gold accents
- Hover animations (scale, color changes)
- Scroll-triggered animations
- Card-based layouts with borders
- Gradient text effects
- Counter animations

### Animation Patterns:
- Fade in on scroll (opacity 0 → 1)
- Slide up on scroll (y: 30 → 0)
- Scale on hover (1 → 1.05)
- Staggered delays for lists
- Smooth transitions (duration: 0.3-0.8s)
