## 🎯 Color Palette Used

### Blue Shades:
```css
blue-900: #1E3A8A  /* Deep blue - borders, primary buttons */
blue-600: #3B82F6  /* Light blue - accents, gradients */
blue-500: #3B82F6  /* Medium blue - icons, focus states */
blue-400: #60A5FA  /* Lighter blue - text, hover states */
```

### Gold (Maintained):
```css
reignGold: #D4AF37  /* Primary gold */
yellow-500: #EAB308  /* Lighter gold for gradients */
```

### Gradients:
```css
/* Blue to Blue-600 */
from-blue-900 to-blue-600

/* Blue-Gold-Blue */
from-blue-400 via-reignGold to-blue-400

/* Gold to Yellow */
from-reignGold to-yellow-500
```

## 🎨 Design Patterns Applied

### Buttons:
1. **Primary CTA**: Blue gradient → Gold gradient on hover
2. **Secondary**: Blue border → Gold fill on hover
3. **Tertiary**: Gold → Blue on hover

### Cards:
- Border: Blue (subtle opacity)
- Hover: Gold border
- Background: Charcoal with transparency

### Text:
- Headlines: Blue-gold gradient
- Accent text: Blue or gold
- Body: White/gray

### Backgrounds:
- Pattern overlays: Blue dots
- Gradients: Blue to gold
- Sections: Black/charcoal base
