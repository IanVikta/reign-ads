# Video Assets

This directory contains video files for the Hero3D component.

## Video Requirements
- Format: MP4 (H.264 codec)
- Size: 1920x1080px (Full HD)
- Duration: 10-30 seconds (looping)
- File size: < 5MB for optimal loading
- Naming: `hero-background.mp4`

## Poster Image
Include a poster image (`hero-poster.jpg`) for video placeholder while loading.

## Current Placeholders
Video file needs to be provided by client.
